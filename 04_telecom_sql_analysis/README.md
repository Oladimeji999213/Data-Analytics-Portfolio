# Telecom SQL Analysis

**Type:** Personal Project
**Tools:** SQL, MySQL, MySQL Workbench

-----

## Project Overview

This project involves the design and querying of a relational database system for a telecommunications company. The database manages customer data, service plans, usage records, billing, and network performance using structured SQL scripts.

The project demonstrates the ability to design database schemas, write complex queries, and extract business insights from relational data.

-----

## Database Structure

The database covers the following key areas:

- Customer and subscription management
- Call, SMS, and data usage records
- Monthly billing and invoicing
- Network performance logging

-----

## Key SQL Concepts Used

- Table creation and schema design
- INSERT statements for data population
- JOIN operations across multiple tables
- COALESCE for handling null values
- GROUP BY for aggregation and summarisation
- Subqueries for advanced filtering
- Business intelligence queries

-----

## Example Business Questions Answered

- What is the billing history for each customer?
- Who are the highest value customers based on total usage?
- What are the call drop and latency rates by network zone?
- What are the monthly service usage patterns across all plans?
- Which customers have exceeded their data limits?
